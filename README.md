# Motivational Countdown & Productivity App

A web-based productivity tool combining a countdown timer, to-do list, and reminders — all wrapped in a sleek, motivational interface.

---

## Features

✨ **Countdown Timer:** Set days, hours, minutes, and seconds; start, pause, reset; audio alert when finished.  
✨ **To-Do List:** Add tasks, mark as done, delete tasks.  
✨ **Reminders:** Schedule reminders with date/time and delete individually.  
✨ **Responsive Design:** Works on desktop and mobile devices.  
✨ **Motivational UI:** Includes an inspiring hourglass image and background.  

---

## Technologies Used

✨ HTML5 & CSS3  
✨ JavaScript (ES6)  
✨ Responsive design with Flexbox  
✨ Audio API for timer notification  

 ## Usage

✨ **Countdown Timer**  
- Input days, hours, minutes, and seconds.  
- Click **Start** to begin the countdown.  
- Use **Pause** to stop temporarily.  
- Click **Reset** to clear the timer.  

✨ **To-Do List**  
- Type a task in the input box.  
- Click **Add** to append it to the list.  
- Click ✔ to mark as done, ✖ to remove.  

✨ **Reminders**  
- Enter a reminder text and select a date/time.  
- Click **Set Reminder** to save it.  
- Remove reminders using the ✖ button.  
 
Customization

✨ Replace wallpaper.jpeg for background.
✨ Replace hourglass.png for the motivational image.
✨ Update script.js audio source for a custom alert sound.
